# 🐹 Go Challenges

A self-paced collection of Go challenges — one concept at a time, increasing in difficulty.
Each quest lives in its own folder with a concept definition and a coding challenge.

## Structure

```
go-quests/
├── 01-variables/
├── 02-loops/
├── 03-conditions/
...
```

Each folder contains:
- `README.md` — concept explanation + the challenge
- `main.go` — your solution goes here
